# 📢 Galaxy Z Flip6 리뷰 분석 및 AI 영상 생성 프로젝트

## 🎯 프로젝트 개요
고객 리뷰 데이터를 활용하여 제품에 대한 감정 분석을 수행하고, 생성형 AI 도구를 활용해 마케팅용 홍보 영상을 제작한 프로젝트입니다.

## 🗓 프로젝트 기간
2025.05 (1주 차 미니 프로젝트)

## 🧑‍💻 주요 역할
- GPT 기반 감정 분석 설계
- 키워드 시각화 및 보고서 작성
- 영상 생성 AI 도구 리서치 및 영상 구성 기획

## 🧰 사용 기술
- ChatGPT / GPTs
- 감성 분석 프롬프트 엔지니어링
- Wordcloud 시각화
- 영상 생성 AI (Sora, Runway 비교 분석)
